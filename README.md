# 🌐 portfolio-dev

Site de portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro.

## ✨ Sobre o projeto

Portfólio criado para apresentar minhas habilidades, projetos e trajetória profissional na área de tecnologia. O design foi desenvolvido com foco em elegância, responsividade e experiência do usuário.

## 🚀 Funcionalidades

- Design responsivo (mobile-first)
- Animações de scroll reveal
- Cursor personalizado
- Seções: Sobre, Skills, Projetos e Contato
- Links diretos para LinkedIn e GitHub
